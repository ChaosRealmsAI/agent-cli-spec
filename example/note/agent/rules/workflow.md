---
name: workflow
description: Step-by-step usage flow
---
1. Run note --help to see all commands
2. note add --title "..." --body "..." to create a note
3. note list to see all notes
4. note search --query "..." to find notes
5. note show <id> to read a specific note
6. note delete <id> --yes to remove (requires confirmation)
7. Report problems with note issue create
